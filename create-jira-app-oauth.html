3. Create the Jira app & configure OAuth permissions

This document covers the steps to register your Q application in Jira and configure OAuth.

A. Jira Cloud — OAuth 2.0 (3LO)

  • Create an app in Atlassian Developer Console.
  • Set callback URL and scopes.
  • Save client ID and secret securely.

B. Jira Server / Data Center — OAuth 1.0a with RSA

  • Generate RSA keypair.
  • Create Application Link in Jira.
  • Configure consumer key and public key.

C. Test the OAuth setup

curl -H "Authorization: Bearer " \
  "https://your-domain.atlassian.net/rest/api/3/issue/PROJ-1"
